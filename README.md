The bot in action



## Install

```bash
pip install -r requirements.txt
```
On MacOS, please also install ```Xcode command line tools``` to run this.

## Run

```bash
python main.py
```

## It can:
- Water plants
- Fertilize plants
- Collect coins
- Play music for them
- Spray them
